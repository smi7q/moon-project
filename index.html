moon

project description

This project is a visually engaging, scroll-responsive web page designed to captivate users with a smooth parallax effect. Developed using HTML, CSS, and JavaScript, it features moving elements such as stars, the moon, mountains, a river, and a boat, all interacting dynamically with the user’s scroll movements to create a rich, immersive experience.
